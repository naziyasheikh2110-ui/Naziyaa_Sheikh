# CSS Sidebar Menu

A responsive **sidebar navigation menu** built with **HTML & CSS**. Ideal for portfolios, art galleries, or content-driven websites.

## Features
- Slide-in sidebar triggered by a hamburger menu
- Menu items with **Font Awesome icons**
- Social media links with hover effects
- Smooth transitions and hover animations
- Full-page background image

## Technologies
- HTML5
- CSS3
- Font Awesome
- Google Fonts (Tinos)

## Usage
1. Clone or download the project.
2. Open `index.html` in your browser.
3. Click the hamburger menu to toggle the sidebar.
